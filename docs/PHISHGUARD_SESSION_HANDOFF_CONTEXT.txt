2) Important bug fixes done
---------------------------

Bug Fix 1: Attachment extraction showed 0 initially
- Root cause: content extraction/popup payload path missing/incomplete.
- Fixed selectors + forwarding path; attachments started appearing in backend logs.

Bug Fix 2: Backend syntax blocker
- `backend/app/core/domain_whitelist.py` had syntax issue.
- Fixed so backend compiles/runs correctly.

Bug Fix 3: Risky-link inconsistency (anchor vs plain text)
- Some emails had risky URLs as plain text (not hyperlink).
- Extraction updated to capture both anchor links and plain-text URLs.

Bug Fix 4: 500 error on email analysis (recent critical)
- Root cause: defanged URLs like `secure-bank-verify[.]example.com` caused `urlparse` ValueError in link analyzer.
- Fix applied:
  - normalized defanged patterns (`[.]`, `(.)`, `{.}` -> `.`)
  - guarded parsing to avoid backend crash
  - applied normalization in main link processing path
- Files changed for this fix:
  - `backend/app/core/link_analyzer.py`
  - `backend/app/main.py`
